二十分钟学习Markdown

发布于: 2011-08-04 16:20
作者: iflat

分类: Python

Markdown 是一种简单的格式语法，它可以非常简单的被工具转换为网页的标记，使用 markdown 能在做文字编辑时将精力集中到文章的内容上，而不是复杂的格式语法上。它的设计哲学就是尽最大可能地方便读、方便写。

 

一、标题的定义

标题定义有两种方法，前缀#号或在标题下加下划线行，形式如下：

#这里是标题

我也是标题 或者 我还是标题

========     ——————————

前缀1个井号是最大的标题，通常被认为是H1标签

前缀2个井号就是二级标题

3个井号是三级标题

四级标题

五级标题
六级

这也是一种标题，我的下一行使用了很多==号

我的下一行使用了很多--号，那就是H2了

 

二、段落的定义

标题是自成一段的，但如果不是标题，使用简单的换行键，markdown 不会产生划分段落的效果。只有2段文字中间有空行时，才标明这是一个新的段落。比如下边这句话，它还是跟我这一段属于同一段落。 “是的，我和上边还是在一个段落里。”

“我才是另起一段。”

如果想在markdown里使用换行效果，简单的方法是在这一行后边敲入2个以上空格，再加一个回车键，这会产生换行。
我是一个新行，注意到了吗？我比段落间距小一点。

还有一种段落叫引用段落，它比正常段落多了一些缩进。引用段落使用>号来产生。

以>打头的被认为是一种引用，通常以缩进的方式展示。你可以在每一行前加>号来将一段文字设为引用段落。 如果每一行都必须加>号，或许有点麻烦，还有一种比较懒的方法是你可以只在一个段落的开始使用>。那么整个段落都是引用格式的，比如我。

引用里可以嵌套其它markdown标记，比如我是引用段落里的一个标题

我是引用里的引用，又缩进了一步。 我是什么，因为没有空行，所以我还是跟上一句是一个段落里的。

多级嵌套的引用也是允许的。

 

三、列表的定义

如果行前有前缀*，-，+，表示这是一个无序列表，例如：

* 我是第一行

* 我是第二行

* 我是第三行

将产生以下列表

  • 我是第一行
  • 我是第二行
  • 我是第三行

或者

- 我是第一行

- 我是第二行

- 我是第三行
将产生同样的一个列表

  • 我是第一行
  • 我是第二行
  • 我是第三行

需要注意的是，文字和前缀符号间要有一个空格。
使用+号的演示：

+ 我是第一行

+ 我是第二行

+ 我是第三行
还是产生同样的一个无序列表

  • 我是第一行
  • 我是第二行
  • 我是第三行

以数字为前缀的行，自然是有序列表，例如：

  1. 我是第一行
  2. 我是第二行
  3. 我是第三行

要注意的是，数字后必须要有句点，和文字间必须再加空格。

其实数字的值在这里没有意义，比如可以这样：

8. 我是第一行

2. 我是第二行

17. 我是第三行
依然将产生从1开始编号的有序列表：

  1. 我是第一行
  2. 我是第二行
  3. 我是第三行

列表标签可以产生段列表，中间可以添加空行，例如：

  • 整个这个段落都是列表，而不管这一段包含了多少文字，是否有换行
    它总是以列表的形式出现。

    我属于上一个列表，只要我跟上一个段落有相同的缩进。

  • 这是段列表的第二项内容。

列表和引用一样，可以有嵌套的markdown内容：

  • 列表里的标题

    列表里的引用

    列表里的引用

  • 无序列表里的代码

        第一行
    第二行
    第三行

为了阻止数字加句点被作为列表，可使用\来转义句点：

1996\. 年 1997\. 年
就不会被当作列表来呈现了，而显示为：

1996. 年 1997. 年

四、代码的定义

正如刚才看到的，行前使用至少4个空格或1个Tab的缩进，可以产生代码格式：

    <div class="footer">
        &copy; 2004 Foo Corporation
    </div>

<div class="footer">
    &copy; 2004 Foo Corporation
</div>

或者：

    import os
    print(os.name)
    print(2**8)

import os
print(os.name)
print(2**8)

如果想呈现代码内的缩进，请在需要缩进的代码前使用双倍到多倍的空格和Tab，例如一次使用8个空格或2个Tab .

    import sys
    for items in sys.__dict__:
        print(items)

将生成为：

import sys
for items in sys.__dict__:
    print(items)

五、水平横条

3个以上的*号或-号，定义一个水平横条。

例如： ***
或----

将呈现水平横条：

或：

当然使用--号时中间要空出一行，不然就是二级标题的定义了。

六、链接的定义

在markdown中用方括号[]来定义链接：

例如：

[百度](https://www.baidu.com)

[百度](https://www.baidu.com , 'title')，这里title是提示语

将产生以下形式的链接：

百度 百度

形如：[About](/about/)的格式，将产生一个到本地的链接：

About定义的是页内链接

形如：[pyper.cn][1]定义的是参考链接

pyper.cn

这样就可以转到 1: http://www.pyper.cn/ "可选的说明文字在这里"

 

如果文字内容直接就是一个可用的URL ,可以使用<xxxx>来直接定义一个链接：

<http://www.pyper.cn/>

用尖括号http://www.pyper.cn/ 定义链接具有同样的效果。

 

七、强调内容的定义

使用1个*或一个_号包围的文字间，产生这种效果：

*我是一段斜体字*

_我也是斜体字_

会显示为：

我是一段斜体字

我也是斜体字

 

如果使用的是2个*号或 _ _ 包围的文字：

**我是加粗字体，这是被2个星号包围的一段内容。**

__好像我也是__
是这种效果：

我是加粗字体，这是被2个星号包围的一段内容。

好像我也是

 

符号`mkdir test`之间的文字是行内代码，行内代码将其包围的内容以原始的样子显现出来，这常常用来呈现行内代码。如果内容本身包含了某些markdown定义的符号，可以使用2个``来将其强调。

`` `2个反引号之间的文字，都被定义为代码` ``

`2个反引号之间的文字，都被定义为代码`

 

八、图片的定义

最后看一下如何在文档内添加图片，图片的定义跟链接非常类似，只不过在方括号前再加上一个感叹号'!'。

![百度logo](https://www.baidu.com/img/bd_logo1.png “百度图片”)

图片定义也可以使用锚点的形式，将需要添加的图片集中放到一块处理。

九、其它定义

同html一样的&copy;版权标志© ；&nbsp;是一个空格， 能看到我吗？

使用\来完成特定符号的转义：

\* 来显示*
\\ 来显示\本身
`` 反引号 `
\_ 下划线_
\{\} 花括号{}
\[\] 方括号[]
\(\) 小括号()
\# 井号#
\+ 加号+
\- 减号-
\. 句点.
\! 感叹号!
