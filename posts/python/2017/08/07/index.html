Pelincan 配置变量

鉴于文档内容超长，将其分2次发布。

以下列举的配置变量其值为其默认值，即当没有在配置文件中设置该变量时，其所取值。

时间和日期 配置变量

TIMEZONE

时区设置。可用的时区请查看：世界时区

DEFAULT_DATE = None

默认日期。如果设置为'fs'，pelican将尝试使用文档的创建日期。

DEFAULT_DATE_FORMAT = '%a %d %B %Y'

默认日期格式

DATE_FORMATS = { }

指定日期格式。注意这里是一个字典，你可以为不同语言设置不同的日期格式。可参考：Python日期文档，如果你的网站使用了多语种，请详细查看。

LOCALE

设置可用的本地化元组，示例：

LOCALE = ('usa', 'jpn', # On Windows
'en_US', 'ja_JP' # On Unix/Linux )

模版 配置变量

TEMPLATE_PAGES = None

自定义模版。可以为特定的源文档，指定一个模版。示例：

TEMPLATE_PAGES = {'src/books.html': 'dest/books.html', 'src/resume.html': 'dest/resume.html', 'src/contact.html': 'dest/contact.html'}

DIRECT_TEMPLATES = ['index', 'categories', 'authors', 'archives']

指定使用那些索引模版。这将为分类和标签等自动建立索引页面。

PAGINATED_DIRECT_TEMPLATES = ['index']

指定那些位置需要分页。默认只是首页有分页，通常我们需要分类也能够分页。

EXTRA_TEMPLATES_PATHS = [ ]

自定义模版的路径。

元数据 配置变量

AUTHOR

默认作者

DEFAULT_METADATA = { }

为元数据设置默认值

FILENAME_METADATA ='(?P<date>d{4}-d{2}-d{2}).*'

提取匹配的元数据信息

PATH_METADATA = ' '

路径元数据信息

EXTRA_PATH_METADATA = { }

额外的路径元数据信息

信息订阅 配置变量

Pelican 默认使用Atom格式的订阅信息，如果你喜欢也可以使用RSS 格式订阅信息。
并且默认是生成分类文章的订阅信息，而不对标签分类进行处理。

FEED_DOMAIN = None, i.e. base URL is "/"

订阅信息的URL，默认为‘/’起点。

FEED_ATOM = None, i.e. no Atom feed

Atom订阅的URL

FEED_RSS = None, i.e. no RSS

RSS订阅的URL

FEED_ALL_ATOM = 'feeds/all.atom.xml'

全部文章的Atom订阅地址

FEED_ALL_ATOM = 'feeds/all.atom.xml'

全部文章的RSS订阅地址

CATEGORY_FEED_ATOM = 'feeds/%s.atom.xml'

Atom分类订阅

CATEGORY_FEED_RSS = None, i.e. no RSS

RSS分类订阅

AUTHOR_FEED_ATOM = 'feeds/%s.atom.xml'

Atom作者订阅

AUTHOR_FEED_RSS = 'feeds/%s.rss.xml'

RSS作者订阅

TAG_FEED_ATOM = None, i.e. no tag feed

Atom标签订阅

TAG_FEED_RSS = None, i.e. no RSS tag feed

RSS标签订阅

FEED_MAX_ITEMS

订阅条目的最大值

RSS_FEED_SUMMARY_ONLY = True

RSS订阅之包含摘要内容

分页 配置变量

Pelican 默认在首页显示所有文章，对文章数量不多的博客来说没有问题，但如果你的博客有上千篇文章，这会是个很不好的体验。

DEFAULT_ORPHANS = 0

默认尾页文章数量。为0表示，不指定最小数量。

DEFAULT_PAGINATION = False

每页文章数量。直接指定数值，为False表示不分页。

PAGINATION_PATTERNS

分页模式。示例：

PAGINATION_PATTERNS = ( (1, '{base_name}/', '{base_name}/index.html'), (2, '{base_name}/page/{number}/', '{base_name}/page/{number}/index.html'), )

多语种 配置变量

DEFAULT_LANG = 'en'

默认语言

TRANSLATION_FEED_ATOM = 'feeds/all-%s.atom.xml'

Atom订阅语言

TRANSLATION_FEED_RSS = None, i.e. no RSS

RSS订阅语言

内容次序 配置变量

NEWEST_FIRST_ARCHIVES = True

归档排序。默认最新归档在前。

REVERSE_CATEGORY_ORDER = False

分类文章排序反转。默认按字母顺序排列。

ARTICLE_ORDER_BY = 'reversed-date'

文章排序方式，默认日期反序

PAGE_ORDER_BY = 'basename'

页面排序方式。

主题 配置变量

THEME

使用的主题名称。示例：

# 使用内置主题 THEME = "notmyidea" # 指定自己安装的主题 THEME = "chunk" # 指定一个自定义主题 THEME = "themes/mycustomtheme" # 使用绝对路径指定一个主题 THEME = "/home/myuser/projects/mysite/themes/mycustomtheme"

THEME_STATIC_DIR = 'theme'

主题目录名称

THEME_STATIC_PATHS = ['static']

主题使用的静态文件的目录名称

CSS_FILE = 'main.css'

自定义样式文件。默认为 main.css

SITESUBTITLE

站点副标题

DISQUS_SITENAME

Disqus帐号

GITHUB_URL

github地址

GOOGLE_ANALYTICS

谷歌统计

GA_COOKIE_DOMAIN

谷歌统计cookie使用的名称

GOSQUARED_SITENAME

MENUITEMS

自定义菜单

PIWIK_URL

Piwik服务器URL

PIWIK_SSL_URL

PiwikSSL URL

PIWIK_SITE_ID

Piwik站点帐号

LINKS

定义一个链接列表，列表元素为(Title, URL)形式的元组。

SOCIAL

同上形式的链接列表。通常两者用来将链接分类。

TWITTER_USERNAME

推特帐号

LINKS_WIDGET_NAME

LINKS链接列表变量的名称

SOCIAL_WIDGET_NAME

SOCIAL链接列表变量的名称
