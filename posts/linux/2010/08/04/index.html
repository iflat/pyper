OpenSuse简介

openSUSE是Novell的开源Linux项目，在2004年2月Novell收购SUSE Linux之后，Novell决定以100%开源产品发布SUSE Linux专业版，旨在推进 Linux 的广泛使用。该项目为 Linux 开发者和爱好者提供了开始使用 Linux 所需要的一切。openSUSE 原名 SUSE Linux，10.2版本以后改名为openSUSE。

openSUSE同时提供了KDE和GNOME作为桌面环境，对个人来说是完全免费的，包括使用和在线更新。OpenSuse最大的贡献就是Open Build Service，它的目的是提供一个稳定的基础并让所有用户可以通过Open Build Service加入到OpenSuse的软件版本维护中来，以得到额外的更多更新的软件。

Novell是一家发兴于上世纪70年代的比较老的软件公司，当年曾凭借Novell NetWare几乎和微软并驾齐驱。在其业务还繁荣的时候收购了SUSE，可惜现在自己都被一家叫做Micro Focus的公司于2014年收购了。

OpenSuse是一个值得试用的Linux发行版，它具有以下一些优点：

  • 符合德国人品质的过硬质量把控
  • 首发滚动发行版，系统和软件可适时更新
  • 易用的YaST软件包管理系统

下载和安装

官方下载站点：https://software.opensuse.org/

可以选择官方下载也可选择在国内镜像站点下载，国内镜像站点不保证最新但下载速度应该有优势(从最近效果来看，官网速度也不错)。常见的国内镜像站点有：

  • 中国科技大学,
  • 网易开源镜像,
  • 阿里云,
  • 搜狐镜像,
  • 清华开源软件镜像站

目前可下载的版本包括：Tumbleweed版 和 Leap版，前者是俗称‘风滚草’的版本，后者是稳定版。Tumbleweed版适合喜欢尝鲜的开发者，Leap版更适合在生产环境中使用。不管你选择了那个版本，其实在安装了以后都可以随时变更为另外一个版本。

不同于Ubuntu，OpenSuse光盘镜像集成了许多常用的软件包，所以大小在4GB以上。例如最新的42.3稳定版，大小为4.7G 。根据你选择的版本，下载到的光盘镜像文件名称类似于：

openSUSE-Tumbleweed-DVD-x86_64-Current.iso
openSUSE-Leap-42.3-DVD-x86_64.iso

现在的安装方式一般是将其制作成可启动U盘，当然你也可以选择将其刻录成光盘来安装。

常见的制作启动U盘的软件有很多，比如：UltraISO 或 Rufus （推荐使用后者，很小巧的工具）。 如果你对命令行比较熟悉，还可以使用dd命令直接将光盘镜像写入U盘。如果你不是在Windows环境下制作启动U盘，请自行百度相应平台的工具。

制作启动U盘的详细操作步骤不再列出，其官网一般都有详细的使用说明。

安装步骤很简单，前提是你对Linux系统要有一个简单的了解。如果你是第一次安装，最好的选择就是将其安装到一个闲置的空白硬盘上。

这是官方的安装教程：OpenSude Installation Quick Start

共有以下简单几个步骤：

  1. 使用制作好的启动U盘或刻录的光盘启动计算机

  2. 选择语言，然后选择安装

  3. 阅读许可协议并同意，点击「下一步」

  4. 设置网络、主机名，如果连接了路由器并使用了DHCP，安装程序会自动网络配置并跳过该步骤

  5. 硬盘分区和挂载点，安装程序会自动推荐一个分区方案。如果你对计算机比较熟悉，可以手动设置分区，但一般情况下推荐的方案也不错。

  6. 选择时区，一般直接点击「下一步」

  7. 选择要使用的桌面环境，KDE和GNOME都不错，还可以两者都选。

  8. 设置用户名和密码

  9. 安装选项：包括引导程序和软件的选择，系统和防火墙设置

  10. 确认并开始安装

  11. 安装过程，将显示详细的安装进度和安装的内容

  12. 安装完成后，计算机将自动重启并引导到新安装的系统中，根据你选择的图形界面，系统将提示你输入密码来进入OpenSuse系统。
