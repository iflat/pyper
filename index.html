  3. 二十分钟学习Markdown

    发布于: 2011-08-04 16:20
    作者: iflat

    分类: Python

    Markdown是一种简单的格式规范，它可以非常简单的被工具转换为网页的标签，使用 markdown 能在做文字编辑时将精力集中到文章的内容上，而不是复杂的格式语法上。

    这里是标题，通常被认为是H1标签

    2个井号就是二级标题

    3个井号是三级标题

    四级标题

    五级标题
    六级，有时候不被某些工具支持

    这也是一种标题，其下一行使用了=

    如果下一行使用了--，那就是H2了

    标题是自成一段的，但如果不是标题，使用简单的换行键，不会换行。只有2段文字中间有空行时，才标明这是一个新的段落。 我和上边还是在一个段落里。

    我才是另起一段。

    以>打头的被认为是一种引用，通常以缩进的方式展示。 如果每一行都必须加>号，或许有点麻烦，那么你可以只在一个段落的开始使用。那么整个段落都是引用格式的。

    引用里可以使用其它标记，比如我是一个标题

    我是引用里的引用，又缩进了一步 我是什么，因为没有空行，所以我还是跟上一句是一个段落里的。

    多级的引用是允许的。

    如果行前有前缀*，-，+，表示这是一个无序列表，例如：

    • 我是第一行
    • 我是第二行
    • 我是第三行

    或者

    • 我是第一行 …
